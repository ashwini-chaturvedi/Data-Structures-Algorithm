## What is Line Sweep?
- It is a Way of Thinking or Visualizing so this is the identification where we have to use line sweep algorithm.
- Line moves in the sorted fashion order i.e. 1,2,3,4 and we will only Focus on what is in line.
  
### Event: In line Sweep ,An Event is a point where somthing changes and you act on it i.e. interval's start or end point, somthing changes and we will work on it.

### Delta: these are values (1/-1) show the changes.

## When to use Line Sweep:
1. Maximum/Minimum view/count of things in Overlapped Intervals.
2. Overlapping Intervals
3. Scheduling Problems
4. Geometrical Shape Problems
5. Area/Skyline Problems.

# Steps:
1.Create 2d Array for (element,1/-1)
2.Conversion:Interval=(start,end)
- (start,1)
- (end,-1)
- the 0th element is called Event and 1st element is called delta.
3.Sort the Event:
- sort the event on basis of 0th element and then 1st element
- In some questions while sorting on basis of 1st element you have to decide in descending or ascending based on if the end time is included or not.
4.Now Traverse the 2d array and then make count and MaxCount or any other thing based on Question.
