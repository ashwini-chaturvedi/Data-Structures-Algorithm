# Pattern 2 - Greedy Interval Selection

## Used When

-   Remove minimum overlaps
-   Keep maximum intervals

## Idea

Sort by ending time.
keep the interval ending earliest

## Pseudocode

``` text
sort by end

take first interval

for every interval:
    if start >= previousEnd:
        keep interval
```
# Time: O(n log n)
```java
public int greedyRemoval(int[][] intervals) {
        Arrays.sort(intervals,(a,b)->a[1]-b[1]);

        int[] prevInterval=intervals[0];
        int count=0;

        for(int i=1;i<intervals.length;i++){
            if(prevInterval[1]>intervals[i][0]){
                count++;
                prevInterval[1]=Math.min(prevInterval[1],intervals[i][1]);//keeping the interval ending earlier
            }else{                
                prevInterval=intervals[i];
            }
        }
        return count;
    }
```
